Finance demo

React native

A dev only front end application created with react native

This project a demo Finance mobile application created by React Native

Purpose

The primary purpose of this application was to learn how to develop dynamic frontend for a mobile phone with react native

Conceptual Design

TBD

Features Implemented

  • Routing: The application can route through different screens
  • Sidebar:The application features a reactive sidebar with dynamic contents
  • Dynamic listView: Lazy loading the list, so it doesnt overpopulate with market values and make the app slow
  • Pinch/Pan Graphs: Graphs can be pinched in or panned to change the viewing scope.

Challenges Faced

So many things to talk about. Like how graphs cannot be natively panned or pinched, we have to introduce changes in the domain to simulate such animations. And so many other things that i chose to skip :3

Future Enhancements

Currently, I do not call it a finished program, as it is only in developer version, and has a lot of bugs

Conclusion

In conclusion, this project showcases my learning procedure with react native
