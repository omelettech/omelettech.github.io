Tic Tac Toe

python

A python game with GUI to learn click and point events

It is a very basic game made in 2020, I was still a beginner so the code is very primitive compared to what logic I would use now. It was still a cool project to get into since I got to know about how a computer reads click and mouse events.

Purpose

To understand the logic behind winning/drawing a game, when to end it and also very simple animations

Conceptual design

The game ends when 

    1. A row, column or a diagonal has the same values
    2. The entire board is filled
after which the an end game status is displayed, and the program halts.
