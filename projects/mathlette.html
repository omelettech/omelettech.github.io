Math graphing calculator

Javascript

A javascript app that lets user draw graphs in the cartesian plane

This program is still work in progress, as I have planned on adding advanced functions such as Derivatives, Integration etc. Yes currently it is simply a poor man's Desmos

Purpose

Main purpose for this design was for user to visualise calculus concepts and possibly other math concepts, while keeping the UI clean and simple. Currently. we can click ON the graph to get the tangent line of the curve ON that point.

Conceptual design

Firstly the input box that recieves user input was heavily validated. making sure it can take as much hit as possible, it is not NEARLY perfect however it still can handle with the user being slightly genorous. Secondly, converting that string into an equation is easier said than done.
I first needed to remove all the redundant spaces and capitalizing, then converted all the operations (+,-,^) into characters that the Math module can handle and displayed the returned results into a script, which takes in all the points and draws the curve for us.

Goals:

Adding extra functionalities, displaying results, adding matrices and more.
